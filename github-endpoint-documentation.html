These queries are for front-end developers who would like to access information from the database
endpoint method body response description
/
/help 		GET none ```ts { endpoints: { '/endpoint': 'description' } } ``` Help Page
/createUser POST ```ts { username: string, password: string, email: string, display_name: string, } ``` ```ts { success: boolean, message?: string | string[], error?: string, status?: number, body?: { username: string, email: string, display_name: string, id: string, } } ``` Create a new user
/authenticate POST ```ts { username: string, password: string, } ``` ```ts { success: boolean, message?: string | string[], error?: string, status?: number, body?: { username: string, email: string, display_name: string, id: string, } } ``` Retrieve user information. Soon to be GET with authentication header.
/changeUsername POST ```ts { username: string, password: string, newUsername: string, } ``` ```ts { success: boolean, message: string, } ``` Change username. This will likely be updated once we have user sessions.
/changePassword POST ```ts { username: string, password: string, newPassword: string, } ``` ```ts { success: boolean, message: string, } ``` Change password. This will likely be updated once we have user sessions.
